Follow these steps to set up the project locally on your machine.

Prerequisites

Make sure you have the following installed on your machine:

- Node.js
- npm (Node Package Manager)
- Git(If you will clone the repo)
- Any archive extractor application(If you will use the zip archive we uploaded)

--------------------------------------------------------------
If you want to clone repo
- git clone https://github.com/barisbatuhanbolat/ozbuild.git
- cd ozbuild
--------------------------------------------------------------

Installation

Install the project dependencies using npm:

- npm install

Running the Project

- npm run dev

Open http://localhost:3000 in your browser to view the project.

Or you can enter our deployed website : https://ozbuild-app.vercel.app
